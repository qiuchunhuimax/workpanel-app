Badge is a small uppercase status pill — for task/project status words (进行中, 已延期), never for counts (use a plain number in a Tag or inline instead).

```jsx
<Badge tone="warning" dot>待复盘</Badge>
<Badge tone="success">已完成</Badge>
```

Always uppercase-styled via CSS regardless of input case; keep label text to 2–4 Chinese characters so it stays scannable in dense lists.
